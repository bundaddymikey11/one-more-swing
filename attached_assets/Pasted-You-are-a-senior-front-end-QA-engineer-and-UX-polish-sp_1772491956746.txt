PHASE 2 — ANCHOR SCROLL FIX (CRITICAL)

Some anchor links are scrolling halfway into sections instead of aligning correctly.

Implement a universal smooth scroll correction:

• If there is a fixed header/navbar:
   - Calculate header height dynamically
   - Offset scroll position properly

Add a global scroll handler that ensures:
   targetElement.getBoundingClientRect().top
   accounts for fixed header height
   and scrolls to exact section top.

If using CSS:
Add:
html {
  scroll-behavior: smooth;
}

If using JS:
Implement a reusable function:

function scrollToSection(id) {
  const element = document.getElementById(id);
  const headerOffset = document.querySelector("header")?.offsetHeight || 0;
  const elementPosition = element.getBoundingClientRect().top + window.pageYOffset;
  const offsetPosition = elementPosition - headerOffset;

  window.scrollTo({
    top: offsetPosition,
    behavior: "smooth"
  });
}

Then refactor all anchor links to use this function.
